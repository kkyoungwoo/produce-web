# Workflow Agent Prompts

이 폴더는 단계별 전문 에이전트 프롬프트를 분리 관리합니다.

- `step1-intake.md`: 콘셉트/화면비율 확정, 콘셉트 변경 시 기본 프롬프트 재초기화
- `step2-script.md`: 주제와 생성 옵션 정리
- `step3-cast.md`: 프롬프트 확인, 대본 생성/수정, 출연자 토글 선택
- `step4-style.md`: 캐릭터 느낌 선택 후 선택 출연자 대표 이미지 확정
- `step5-handoff.md`: 프로젝트 저장 및 씬 제작 진입
- `final-scene-production.md`: 씬 생성부터 최종 영상 제작

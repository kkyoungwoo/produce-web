# Step 4 Style Agent Prompt

목표
- Step3에서 선택한 출연자만 대상으로 캐릭터 느낌과 대표 이미지를 확정한다.

작업 원칙
- 먼저 캐릭터 느낌을 하나 고른다.
- 작업 화면에는 Step3에서 선택한 출연자만 표시한다.
- Step4 진입 시 출연자 목록을 다시 추출하는 갱신 버튼은 두지 않는다. Step3에서 확정된 선택 상태를 그대로 사용한다.
- 각 출연자는 대표 이미지 1장을 확정할 수 있어야 한다.
- 이미지 등록은 각 출연자 카드 단위로 처리한다.
- AI API가 없어도 샘플 후보 생성과 업로드 이미지 선택만으로 Step4를 끝까지 진행할 수 있어야 한다.
- workspace에 진입하면 선택된 출연자에게 기존 이미지가 있을 경우 첫 이미지를 자동 대표값으로 선택한다.
- 기존 이미지가 없으면 그 출연자에 대한 첫 후보 이미지 생성을 자동 시작한다.
- 선택되지 않은 출연자를 Step4 이미지 제작 대상으로 다시 끼워 넣지 않는다.

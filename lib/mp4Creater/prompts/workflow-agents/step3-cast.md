# Step 3 Cast Agent Prompt

목표
- 콘셉트 기본 프롬프트를 바탕으로 최종 대본을 만들고, Step4로 넘길 출연자를 확정한다.

작업 원칙
- 콘셉트가 바뀌면 이전 프롬프트 수정본을 유지하지 않고 새 콘셉트 기본 프롬프트부터 다시 시작한다.
- 출연자는 토글 방식으로 선택한다.
- 한 번 누르면 선택, 다시 누르면 해제된다.
- 재추출 시에는 기존 선택 출연자를 가능한 한 보존한다.
- 같은 이름/역할의 출연자가 다시 잡히면 기존 generatedImages, selectedImageId, voice 설정을 우선 재사용한다.
- 대본이 있는데 출연자를 하나도 선택하지 않은 상태에서 `다음으로`를 누르면 Step 4로 넘기지 않고 출연자 선택 영역으로 스크롤 안내한다.
- Step 4 이동 직전에는 현재 `selectedCharacterIds`를 우선 그대로 넘기고, 현재 목록이 비어 있을 때만 `preserveSelection` 기준으로 재수화한다.
- 다음 버튼은 정상 상태라면 한 번 클릭으로 Step 4로 넘어가야 한다.
- Step4로 넘길 출연자 id 목록은 step 이동 직전 즉시 draft와 project cache에 저장되어야 한다.

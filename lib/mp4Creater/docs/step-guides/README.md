# mp4Creater step guide

이 폴더는 Step별 수정 경로를 빠르게 찾기 위한 짧은 안내 모음입니다.

## 공통 규칙
- 해당 Step 기능이 바뀌면 **이 md와 실제 파일을 함께 수정**합니다.
- 저장은 넉넉하게, 생성 참조는 선택본 위주로 다룹니다.
- UI보다 데이터 계약과 저장 구조를 먼저 확인합니다.
- mp4Creater 외 경로는 꼭 필요한 경우만 수정합니다.

## 추천 흐름
1. 해당 Step md 읽기
2. 적힌 경로의 컴포넌트와 서비스 확인
3. 저장/export/import 연결 파일도 같이 확인
4. 수정 후 관련 md도 바로 갱신

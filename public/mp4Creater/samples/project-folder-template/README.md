# Project Folder Template

실제 프로젝트가 저장되면 아래 구조가 자동 생성됩니다.

```
projects/
  project-0001-your-project/
    project.json
    metadata/
    prompts/
    images/
    videos/
    audio/
    thumbnails/
    characters/
    styles/
```

- `project.json` : 프로젝트 전체 상태
- `metadata/` : 씬/워크플로우 요약 JSON
- `prompts/` : 대본/씬/캐릭터/썸네일 프롬프트
- `images/`, `videos/`, `audio/` : 결과 파일
- `characters/`, `styles/`, `thumbnails/` : 선택/후보 이미지

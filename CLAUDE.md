# CLAUDE.md

이 문서는 토큰을 아끼면서 안정적으로 일하도록 맞춘 운영 규칙입니다.

## 1) 컨텍스트 로딩
- 한 번에 모든 문서를 읽지 않습니다.
- 먼저 `AGENTS.md`, `agent.md`, 작업 대상 전용 문서만 읽습니다.
- 코드 파일은 실제 수정 대상만 최소 범위로 읽습니다.
- `.next`, `dist`, `coverage`, 대용량 로그는 기본적으로 읽지 않습니다.

## 2) mp4Creater 프롬프트 정책
- 사용자 입력 언어는 그대로 유지합니다.
- AI 연결 직전의 프롬프트만 영어로 번역합니다.
- 번역 캐시를 사용해 동일 문장에 토큰을 반복 사용하지 않습니다.

## 3) 문서 갱신 규칙
- 기능이 업데이트되면 관련 MD도 같은 작업에서 함께 갱신합니다.
- 과거 기준 설명서, 임시 메모, 중복 MD는 남겨두지 않고 삭제하거나 최신 문서로 통합합니다.
- 에이전트는 오래된 MD를 참고문서로 계속 누적하지 않고, 현재 동작과 맞는 문서만 유지합니다.
